# Local Business Landing Page & Digital Menu

A lightweight, static web presence developed for a local business to provide customers with easy access to their digital menu via QR code.

🚀 **Live Site:**  [harabizaga.com](https://harabizaga.com)

## Tech Stack
* **Frontend:** HTML, CSS
* **Hosting & Delivery:** Vercel (Automated deployments directly from GitHub)

## Project Context
* **Problem:** The business needed a cost-effective, easily accessible digital menu without the overhead of a complex ordering system.
* **Solution:** Developed a highly optimized static landing page that directs users to the most up-to-date PDF menu. 
* **Outcome:** Eliminated printing costs for menu updates and provided a seamless mobile experience for customers.
